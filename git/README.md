`右键git bash here`
//打开文件

`git init `
//初始化git仓库

`git add first.txt` 
//添加到仓库

`git commit -m "这是第一次提交" `
//提交

`git remote add origin 地址链接 `
//添加到远程仓库

`git push -u origin master `
//推到远程仓库

`git pull origin master`
//拉到本机

`git clone 地址链接 `
//克隆到本地

`git diff `
//改了啥

`git status `
// 状态

`git pull origin master --allow-unrelated-histories  ` 
// :Q退出
// 把远程仓库和本地同步，消除差异

`git remote rm origin`
删除远程分支
